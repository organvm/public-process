# Community

Public Process is the discourse layer for the organvm system. The essays are
published as a feed, but the work is meant to be inspected, questioned, and
improved in public.

## Where to Talk

- **Questions and discussion**: Open a
  [Question / Discussion issue](https://github.com/organvm/public-process/issues/new?template=question_discussion.md)
  for open-ended questions about the essays, methodology, or organvm system.
- **Stranger-test feedback**: Use the
  [Stranger Test Feedback template](https://github.com/organvm/public-process/issues/new?template=stranger_test_feedback.md)
  when you are recording first-time-reader friction.
- **Bugs and improvements**: Use
  [GitHub Issues](https://github.com/organvm/public-process/issues) for site
  defects, broken links, documentation gaps, and concrete feature requests.
- **Author contact**: Reach the author through
  [@4444J99 on GitHub](https://github.com/4444J99) or by opening a public issue
  in this repository.

There is not a Discord or Slack workspace for this repository right now. Public,
asynchronous discussion happens on GitHub so questions, decisions, and follow-up
work stay attached to the process record.
